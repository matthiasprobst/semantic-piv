- Every prefixed name you use (e.g., piv:Camera) MUST have a corresponding @prefix declaration in YOUR output.
- Do NOT use undeclared prefixes.

- Output ONLY raw Turtle instance triples (+ required @prefix lines).

- If ALLOWED TERMS are provided, use ONLY those classes/properties/datatypes.

- Assert ONLY facts explicitly stated in TEXT INPUT.
- No guessing, inference, normalization, or unit conversion.

5) Identifiers

- If IRIs are minted, follow the BASE_URI rules from the USER prompt.

6) Literals

  (a) the prov: prefix is declared in your output, AND

8) Minimality
- Emit the minimum triples needed to encode extracted facts.
- Avoid duplicate or redundant triples.
